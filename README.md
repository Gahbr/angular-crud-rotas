# ✔️ Lista de usuarios em Angular


## 💻 Sobre o projeto
Crud de lista de usuários feita em Angular e Bootstrap. Para salvar os dados foi feita a conexão com a API da sheetBest


## 🛠 Tecnologias utilizadas
* Angular
* Typescript
* Bootstrap 
* HTML
* CSS
